# Attributes dan Methods
### Class Kategori
    Atribut:
    * id_kategori (string)
    * nama_kategori (string)
    * deskripsi (string)

    Methods:
    1. Getter
       Untuk mengambil nilai atribut
    2. Setter
       Untuk mengubah nilai atribut

### Class Barang
    Atribut:
    * merk (string)
    * model (string)
    * tahun_produksi (int)

    Methods:
    1. Getter
       Untuk mengambil nilai atribut
    2. Setter
       Untuk mengubah nilai atribut

### Class DetailBarang
    Atribut:
    * nama_barang (string)
    * harga (double)
    * stok (int)

    Methods:
    1. Getter
       Untuk mengambil nilai atribut
    2. Setter
       Untuk mengubah nilai atribut

### Main
    Methods:
    1. tampilkanMenu
       Untuk menampilkan menu program
    2. hitungLebarKolom
       Untuk menghitung lebar kolom sesuai size suatu atribut (tabel dinamis)
    3. tampilkanData
       Untuk menampilkan seluruh data
    4. tambahData
       Untuk menambah data sesuai input dari user
